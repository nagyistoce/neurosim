Currently supports only Win7 x86_64

Install AMD APP SDK
From here 2 ways:

1)
  Insall Visual Studio Express or Pro
  Install Strawberry Perl
  Install msys
  Configure MSWin32 node in config/config.xml file
  Run neurosim.pl

2)
  Insall Visual Studio Pro
  Open VS Solution file
  Build SDKUtil project
  Build IntegrationTest project
  Run IntegrationTest
  Find binaries in neurosim\cpp_cl\app\Neuro\bin
